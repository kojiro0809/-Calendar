# Store Calendar Generator - 実店舗向けカレンダー自動生成ツール (Python/Tkinter)

スマートフォンのテンプレートアプリで毎月数時間かけていた「店舗の営業日カレンダー作成業務」を、PC上で数クリック・数秒で完結させるために開発したデスクトップアプリケーションです。
![Uploading レコーディング-2026-04-10-110929.gif…]()



## 開発の背景：身近な「非効率」
私の母は実店舗を経営しており、毎月の営業日や定休日、ワークショップ（WS）の予定を告知するためのカレンダー画像を作成していました。
しかし、スマホアプリのテンプレートを使用しているにもかかわらず、**「毎月すべての日付を手作業で調整し直す」「スタンプを1つずつ配置する」**という非常に非効率な作業が発生し、毎月多大な時間を奪われていました。

「この定型作業は、プログラミングの力で一瞬で終わらせられるはずだ」と考え、母（＝実際の顧客）の業務要件をヒアリングし、本ツールの開発に着手しました。

## プロダクトのこだわりと「アジャイルな改善」

### 1. 非エンジニアが「すぐ使える」デリバリー設計
開発したツールは、Python環境がない母のPCでも即座に導入できるよう、実行可能ファイル（`.exe` / `.app`）としてビルドして納品しました。**「開発して終わり」ではなく「エンドユーザーが迷わず使える状態にする」**というデリバリーの責任を意識しました。

### 2. 実ユーザーからのフィードバックとUI/UX改善
初期バージョン（MVP）を母に使ってもらった際、「作業は一瞬で終わって感動したけれど、カレンダーのデザインがダサい」という非常にリアルで率直なフィードバックを受けました。
そこで、以下のリファクタリングを実施しました。
- **カラーパレットの刷新:** 原色に近い色合いから、店舗のSNSに投稿しても違和感のないモダンな配色（テラコッタやセージグリーン）へ定数を変更。
- **視認性の向上:** PIL (Pillow) の描画ロジックを見直し、グリッド線の色を和らげることで「抜け感」のあるデザインに修正。

### 3. 誤操作防止（フェイルセーフ）の実装
画像のエクスポート時に意図しない上書きを防ぐための確認ダイアログや、月をまたいだ無効な日付の自動非表示など、非エンジニアが直感的に触ってもエラーが起きないよう例外処理を組み込んでいます。

##  技術スタック
- **Language:** Python 3.x
- **GUI Framework:** Tkinter
- **Image Processing:** Pillow (PIL)
- **Deployment:** PyInstaller (スタンドアロン化)

このプロジェクトを通じて、「技術的に高度なこと」よりも**「特定の誰かの苦痛を取り除き、時間を生み出すこと」こそがエンジニアリングの真の価値である**と実感しました。
